RENDERPASSES:
- GBufferRenderPass				-> 1
	read: none
	write:
		- gbuffer image (1920x1080)
- IndirectLightRenderPass		-> 1
	read:
		- gbuffer image
		- previous gbuffer image
		- previous indirect light image
	write:
		- indirect light image (1280x720)
- LightRenderPass               -> 1
	read:
		- indirect light image
		- gbuffer image
	write:
		- light image (1920x1080)
- PostProcessRenderPass         -> 1
	read:
		- light image
	write:
		- swapchain image (2560x1440)

DESCRIPTORSETS:
- [0] LowFrequencyDescriptorSet		   -> 1 (using dynamic offsets)
- [1] HighFrequencyDescriptorSet	   -> 0 (currently not used)
- [2] TextureAttachmentsDescriptorSets
	- IndirectLightPassDescriptorSet   -> 2
		- gbuffer image
		- previous gbuffer image
		- previous indirect light image
	- LightPassDesciptorSet            -> 2
		- indirect light image
		- gbuffer image
	- PostProcessDescriptorSet		   -> 2
		- light image
	- DebugDescriptorSet			   -> 2
		- gbuffer images
		- indirect light image
		- light image
